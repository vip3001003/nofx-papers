# NOFX Paper Radar

自动抓取 arXiv 量化金融论文，每日更新，服务于 NOFX 量化交易系统研究。

## 使用

直接查看 [PAPERS.md](PAPERS.md)，用 `Ctrl+F` 搜索关键词或分类标签。

## 分类

| 标签 | 内容 |
|------|------|
| `momentum` | 动量/均值回归/Alpha因子 |
| `regime` | 市场制度检测/HMM/波动聚类 |
| `ml` | 机器学习/深度学习/RL交易 |
| `volume` | 订单流/成交量/微观结构 |
| `risk` | 回撤/风险管理/仓位优化 |
| `crypto` | 加密货币/比特币/数字资产 |

## 更新频率

GitHub Actions 每天 UTC 02:00（北京时间 10:00）自动运行，有新论文则提交。
