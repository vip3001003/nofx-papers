# NOFX Paper Radar

每日自动抓取 arXiv 量化金融论文，按相关度排序。

**分类说明：**
- `momentum` — 动量/均值回归/Alpha因子
- `regime` — 市场制度检测/隐马尔可夫/波动聚类
- `ml` — 机器学习/深度学习/强化学习用于交易
- `volume` — 订单流/成交量/市场微观结构
- `risk` — 回撤/风险管理/Kelly/仓位优化
- `crypto` — 加密货币/比特币/数字资产

**相关度评分**：命中NOFX核心关键词数量（IC/factor/regime/MACD/Sharpe等），分数越高越相关。

**文件夹说明：**
- `abstracts/` — 英文原文（供AI分析）
- `abstracts_zh/` — 中文翻译（供用户阅读）

**搜索方式：** `Ctrl+F` 关键词，或 GitHub 页面右上角搜索。
