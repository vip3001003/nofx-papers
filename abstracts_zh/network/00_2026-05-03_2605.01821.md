# 局部最优网络上相邻可能的创新动态

- **arXiv**: [2605.01821](https://arxiv.org/abs/2605.01821)
- **日期**: 2026-05-03
- **分类**: network
- **作者**: Leonardo Rizzo, Edward D. Lee, János Kertész
- **NOFX相关度**: 0

## 摘要

我们提出局部最优网络（LON）作为创新动态建模的正式框架。 LON 是一个有向加权图，其中节点代表局部稳定的技术配置，边编码其吸引力盆地之间的转移概率。我们从适应度景观构建 LON，并将创新代理建模为随机步行者，探索所得网络上的相邻可能。我们证明，该模型同时生成发现过程传统的四个主要经验规律：次线性新颖性增长（堆定律）、重尾频率分布（齐普夫定律）、反常波动尺度（泰勒定律）和幂律分布事件间时间。指数落在经验观察的范围内，并受到 LON 拓扑的共同约束。 LON 的社区提供了基于流域级可达性的技术范式的操作定义。因此，LON 框架将创新建模的发现过程和自适应搜索传统连接到一个单一的、简约的、可凭经验测试的表示中。
