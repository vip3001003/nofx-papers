# 真实生产网络上的类霍尔横向应力和沙堆临界性

- **arXiv**: [2605.01561](https://arxiv.org/abs/2605.01561)
- **日期**: 2026-05-02
- **分类**: network
- **作者**: Diego Vallarino
- **NOFX相关度**: 3

## 摘要

本文开发了一种经济不稳定的霍尔沙堆模型，该模型将类霍尔横向应力机制与真实生产网络基底上的沙堆阈值动力学相结合。与物理霍尔效应类似，外部场下的暴露流会产生横向应力，我们将经济冲击建模为作用于流密集、低冗余、低容量节点的场，并通过乘法转换函数产生系统应力。累积的应力驱动离散的倾倒规则和雪崩动力学，其有效激活阈值随着横向暴露而下降。该模型在 2000--2014 年年度世界投入产出数据库 (WIOD) 生产网络上进行校准，并在 2014 年基底（2{,}283 个国家/地区节点）上进行模拟，采用三种替代传播标准化，以避免行随机算子的机械近临界性。关于外部场强度和冗余应力的受控蒙特卡罗实验产生四种有序状态：稳定吸收、潜在脆性、临界转变和雪崩状态。平均雪崩大小和有限大小系统事件的概率 $\Pr(S\!\geq\!5)$、$\Pr(S\!\geq\!10)$ 和 $\Pr(S\!\geq\!20)$ 与场强度和冗余应力共同上升。尾部诊断显示雪崩分布的机制依赖性增厚，但估计的尾部指数仍然太高，无法解释为普遍幂律临界性的证据。因此，本文的贡献是对横向应力如何激活结构脆弱性的有限规模的真实网络描述，而不是对全球经济中自组织临界性的主张。
