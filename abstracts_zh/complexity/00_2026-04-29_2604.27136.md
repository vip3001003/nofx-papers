# 通过 EATR-flooding 加强速率计算

- **arXiv**: [2604.27136](https://arxiv.org/abs/2604.27136)
- **日期**: 2026-04-29
- **分类**: complexity
- **作者**: Nicodemo Mazzaferro, Willmor J Pena Ccoa, Pilar Cossio et al.
- **NOFX相关度**: 0

## 摘要

最近的几种方法表明，可以使用模拟来计算非常慢的生物分子过程的速率常数，其中沿着一个或多个集体变量（CV）添加依赖于时间的偏差。我们之前报道了指数平均时间相关速率 (EATR) 方法，该方法可以通过考虑外部偏置电位如何有效地使用学习的 CV 质量因子 $γ$ 修改观察到的速率来改进这些方法。当偏置次优坐标时，这会导致使用相同的数据进行更准确的速率估计。然而，所制定的 EATR 依赖于随时间变化的偏置电位来正确确定偏置效率，这限制了该方法对准静态偏置方案的适用性，例如“泛洪”或动态概率增强采样 (OPES)。在这里，我们提出了 EATR-flooding 方法，该方法通过在多组模拟中改变（增强）偏置电位的强度来取代对时间相关偏置的需要，从而概括了我们的方法。我们将这种方法实现为开源 Python 库，并证明与粗粒度蛋白质系统的标准 EATR 相比，这种方法是准确的，不会造成明显的效率损失，并且在完全原子空腔配体模型上也表现出良好的性能。 EATR-flooding 的另外两个吸引人的功能是对过度偏置的内部检查，以及对于给定的 CV 选择仅预测单个 $γ$ 参数这一事实，与我们之前的结果相比，$γ$ 凭经验取决于偏置率。最后，我们相信 EATR-flooding 不仅适用于 OPES 模拟，更普遍地适用于 CV 偏置增强采样方法，使其具有广泛的用途。
