# 带电胶体晶体中非 DLVO 相互作用驱动的静电弹性软化和紫外线不稳定性

- **arXiv**: [2604.22448](https://arxiv.org/abs/2604.22448)
- **日期**: 2026-04-24
- **分类**: complexity
- **作者**: Hao Wu, Zhong-Can Ou-Yang
- **NOFX相关度**: 2

## 摘要

被移动离子渗透的胶体晶体表现出静电自由度和弹性自由度之间的耦合，从而重新规范化有效屏蔽长度并引起波矢量相关的弹性软化。基于最近提出的连续模型 [\textit{Commun.理论。 Phys.} \textbf{77}, 055602 (2025)]，我们进行了严格的高斯波动分析来阐明均质相的稳定性极限。通过积分静电涨落，我们得出有效弹性模量 $Γ(q)$ 作为波矢 $q$ 的函数。我们表明，在完美离子屏蔽的保护下，长波长模量 $Г(0)$ 仍然等于裸模量 $βK$。相比之下，短波长模量 $Г(q\to\infty) = βK(1-Ψ)$ 随着静电弹性耦合 $Ψ\equiv 2βn_0 v_0^2 K$ 的增加而软化，在临界值 $Ψ=1$ 处消失。对于$ xi > 1 $，涨落谱对所有波矢量$ q > q_c = κ_0 / \ sqrt { xi-1} $表现出负特征值，表明均匀相的紫外不稳定性。在真实的胶体晶体中，这种发散由离散晶格截止 $q_{\max}\simπ/a$ 调节，将物理不稳定性限制在有限带 $q_c < q < q_{\max}$ 内。宏观极限$q\to 0$对于所有$Ψ$保持无条件稳定。因此，$xi=1$ 处的转变标志着短波长机械故障的开始，而宏观弹性刚度保持不变。我们的分析阐明了最小耦合模型的正确物理解释，并提供了非 DLVO 相互作用如何驱动带电胶体晶体中局部结构塌陷的一致图景。
