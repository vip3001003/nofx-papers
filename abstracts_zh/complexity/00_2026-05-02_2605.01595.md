# 非齐次哈密顿量淬灭后的纠缠动力学

- **arXiv**: [2605.01595](https://arxiv.org/abs/2605.01595)
- **日期**: 2026-05-02
- **分类**: complexity
- **作者**: Andrea Di Pasquale, Federico Rottoli, Vincenzo Alba
- **NOFX相关度**: 0

## 摘要

我们研究了由 $H = H_L + H_R$ 形式的非齐次哈密顿量控制的二分系统中的纠缠动力学，其中 $H_{L/R}$ 仅作用于左侧或右侧区域，并且在每个区域内是均匀的。针对XX链和横向场伊辛链，我们推导了长时间流体力学极限下两个区域之间的纠缠熵的解析公式。在这种情况下，入射到界面上的费米子会发生散射，从而在反射模式和透射模式之间产生纠缠。由此产生的准粒子图像由传输系数控制，我们通过求解稳态晶格薛定谔方程来解析获得该系数。由于有界色散，强烈的不均匀性抑制了传输和缠结的增长。我们将我们的分析预测与范式设置中的数值模拟进行基准比较。最后，我们使用 tDMRG 将分析扩展到相互作用的 XXZ 链。数值数据显示与二次情况的定性一致性：纠缠增长在强非均匀极限下仍然受到抑制。然而值得注意的是，即使在传输受到抑制时，至少在中间时间，缠结也会继续增加。
