# 空腔介导的局域化和集体电子关联相

- **arXiv**: [2605.01551](https://arxiv.org/abs/2605.01551)
- **日期**: 2026-05-02
- **分类**: complexity
- **作者**: Dominik Sidler, Michael Ruggenthaler, Angel Rubio
- **NOFX相关度**: 0

## 摘要

分子系团与光腔的集体强耦合开辟了一条通过真正集体电子关联来改变物质的途径。然而，即使没有空腔，库仑相关性也很难描述，而且空腔耦合增加了延伸到整个分子系综的横向相关通道。在这里，我们表明，通过将集体分子间电子相关性映射到可分析求解的球形谢林顿-柯克帕特里克模型，这个看似棘手的问题可以进行受控描述。由此产生的理论预测了两个集体相关相，即副相关相和自旋玻璃相关相，超出了传统的不相关分子状态。这些相揭示了一种熵驱动的局域-离域机制，可将分子电子态转变为集体相关的腔态。我们的工作将空腔介导的电子相关性建立为强耦合分子系综中涌现相的微观机制。
