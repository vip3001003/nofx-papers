# JaxMARL-HFT：用于高频交易的 GPU 加速大规模多代理强化学习

- **arXiv**: [2511.02136](https://arxiv.org/abs/2511.02136)
- **日期**: 2025-11-03
- **分类**: ml
- **作者**: Valentin Mohl, Sascha Frey, Reuben Leyland et al.
- **NOFX相关度**: 2

## 摘要

用于高频金融市场的基于代理的建模（ABM）方法很难校准和验证，部分原因是定义固定代理策略所创建的参数空间很大。多智能体强化学习（MARL）可以实现更真实的智能体行为并减少自由参数的数量，但迄今为止繁重的计算成本限制了研究工作。为了解决这个问题，我们引入了 JaxMARL-HFT（基于 JAX 的高频交易多智能体强化学习），这是第一个 GPU 加速的开源多智能体强化学习环境，适用于按订单 (MBO) 数据进行高频交易 (HFT)。 JaxMARL-HFT 扩展了 JaxMARL 框架并建立在 JAX-LOB 实现的基础上，旨在处理异构代理集，从而实现不同的观察/操作空间和奖励功能。它设计灵活，因此也可以用于单代理强化学习，或扩展为具有固定策略代理的 ABM。与相同硬件上最先进的参考实现相比，利用 JAX 可以将端到端训练时间缩短高达 240 倍。这种显着的加速使得利用高频交易中可用的大型、精细数据集以及执行交易中稳健且高效的 MARL 研究所需的广泛超参数扫描成为可能。我们使用一年的 LOB 数据（4 亿个订单）演示了 JaxMARL-HFT 与独立的近端策略优化 (IPPO) 在两人环境中的使用，其中包括订单执行和做市代理，并表明这些代理学习超越标准基准。 JaxMARL-HFT 框架的代码可在 GitHub 上获取。
